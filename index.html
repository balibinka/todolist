How to use this table

  • Each row represents a day of the week, from Monday to Sunday.
  • Each column represents a task slot for that day, from Task 1 to Task 5.
  • To add a task you need to click on an empty cell and write your task.
  • When you add your task you can mark it
  • Click the left mouse button on your task to marked it as completed.
  • And click on it right to remove it.
